# The-Quick-Commerce-Face-Off-Comparative-Analysis-of-Blinkit-JioMart-and-Instamart
Q-Comm Analysis: Blinkit, JioMart &amp; Instamart. End-to-end data analytics project. Performed ETL with Pandas to clean retail data, modeled and ran EDA in MySQL to identify pricing trends, and built an interactive Power BI dashboard to visualize delivery performance and market share across platforms.
